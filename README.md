## Quick Start

### Prerequisites

  * [NodeJS](https://nodejs.org)
  * [Docker](https://www.docker.com/products) - Uses `docker-compose` for coordination

### Installation

Run the following in a new folder

```
npm install -g yo generator-edge
yo edge:server
```

### Starting and Stopping

```
npm start
npm restart
npm stop
```
